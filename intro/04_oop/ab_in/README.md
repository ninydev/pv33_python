# Приклад ООП: Абстрактні методи та інтерфейси (Монітори та Комп'ютери)

Цей проект демонструє використання основних принципів ООП у Python: абстракції, інтерфейсів (через множинне успадкування) та патерну "Адаптер".

## Структура проекту

- `base.py`: Містить абстрактні базові класи `Monitor` та `Computer`. Вони визначають загальний контракт: монітор має приймати сигнал, комп'ютер — передавати.
- `interfaces.py`: Визначає інтерфейси `HDMI` та `VGA`. У Python вони реалізовані як абстрактні класи.
- `monitors.py`: Конкретні реалізації моніторів (`SamsungMonitor`, `OldDellMonitor`).
- `computers.py`: Конкретні реалізації комп'ютерів (`MacBook`, `OldPCLaptop`).
- `adapter.py`: Реалізація адаптера `VGAToHDMIAdapter`, який дозволяє підключити старий комп'ютер до нового монітора.
- `main.py`: Приклади використання та демонстрація взаємодії об'єктів.

## Як це працює

1. **Абстракція**: Ми не знаємо точно, як працює кожен монітор, але ми впевнені, що він має метод `receive_video_stream`.
2. **Інтерфейси**: За допомогою інтерфейсів ми вказуємо, які саме роз'єми підтримує пристрій.
3. **Адаптер**: Дозволяє об'єктам з несумісними інтерфейсами працювати разом.

## Запуск

Щоб запустити приклад, виконайте:

```bash
python3 -m intro.04_oop.ab_in.main
```
