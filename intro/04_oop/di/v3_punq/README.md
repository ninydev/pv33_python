# Punq Dependency Injection

`punq` — це легковісна і дуже проста бібліотека для впровадження залежностей (IoC контейнер) в Python, яка спирається на анотації типів (type hints). Вона ідеально підходить для невеликих та середніх проектів.

## Встановлення модулів

Щоб встановити бібліотеку, виконайте команду:
```bash
pip install punq
```

## Як це працює

1. Створюється екземпляр контейнера: `container = punq.Container()`.
2. Відбувається **реєстрація (Register)** класів та інтерфейсів:
   - `container.register(StorageInterface, LocalStorage)`: ми вказуємо контейнеру, що кожного разу, коли хтось просить `StorageInterface`, йому треба віддати об'єкт `LocalStorage`.
   - `container.register(AvatarService)`: реєструємо сам сервіс. Оскільки в його `__init__` вказано `storage: StorageInterface`, `punq` сам зрозуміє за допомогою рефлексії, що туди треба підставити зареєстроване сховище.
3. **Отримання (Resolve)**:
   - Метод `container.resolve(AvatarService)` автоматично створює і повертає готовий екземпляр `AvatarService` зі всіма впровадженими в нього залежностями.

## Запуск

Просто запустіть файл:
```bash
python main.py
```
